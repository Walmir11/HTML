Tags HTML

Este guia rápido apresenta uma visão geral das principais tags HTML utilizadas para estruturação e formatação de conteúdo em páginas da web. Desde títulos e parágrafos até listas e links, estas tags são fundamentais para a construção de uma página HTML bem organizada e acessível.

  • Texto
  • Listas
  • Formatação de texto
  • Referencias

Títulos

(Voltar)
  • 'h1' até 'h6': Definem títulos, do mais importante 'h1' ao menos importante 'h6'.

Texto

(Voltar)
  • 'p': Define um parágrafo.
  • 'mark': Destaca o texto com marcação amarela.
  • 'small': Define texto menor.
  • 'i': Texto em itálico.
  • 'u': Texto sublinhado.
  • 'strong': Texto em negrito, indicando importância.

Listas

(Voltar)
  • 'ol': Define uma lista ordenada (numerada).
  • 'ul': Define uma lista não ordenada (com marcadores).
  • 'li': Define um item de lista.

Formatação de texto

(Voltar)
  • 'sub': Define texto em subscrito.
  • 'sup': Define texto em sobrescrito.
  • 'blockquote': Define uma citação em bloco.
